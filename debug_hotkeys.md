## 常规

|  快捷键  |              说明               |
| :------: | :-----------------------------: |
|   `R`    |    返回到存档界面 / 重启游戏    |
|   `O`    |         限制帧率到 3fps         |
|   `P`    |        提高帧率到 60fps         |
|   `L`    |            加载存档             |
|   `S`    |            保存存档             |
| `退格键` | 跳转到 Kris 房间（在命名界面）  |
|   `F`    | 提高帧率到 58fps （貌似不管用） |
|  `F10`   |        截图到存档目录下         |
